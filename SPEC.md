## 対応データ

### CSV

松井証券などの歩み値CSVを読み込みます。

代表形式:

```csv
値段,株数,金額,時刻
5762,1812400,10443048800,15:30:00
5791,1500,8686500,15:24:59
```

仕様:

- `値段` または `時刻` を含むヘッダーをQR/歩み値形式として判定します。
- `値段`, `価格`, `price` を価格列として扱います。
- `株数`, `出来高`, `volume`, `vol` を数量列として扱います。
- `時刻`, `時間`, `time` を時刻列として扱います。
- `日付`, `年月日`, `date` があれば行の日付として使います。
- ファイル名に `YYYYMMDD` があれば取引日として使います。
- ファイル名に `6269`, `285A` などの銘柄コードがあれば銘柄名表示に使います。
- 複数CSV選択時は時刻順に結合します。
- BOM付きUTF-8、カンマ区切り、単純なクォート付きCSVに対応します。

内部形式は次です。

```js
{ time: unix秒UTC, price: Number, volume: Number }
```

### 複数銘柄JSON

次のような歩み値JSONに対応します。

```json
{
  "map": { "p": "price", "t": "timestamp", "f": "frame", "q": "quantity", "k": "kind" },
  "ticks": {
    "9984": [
      { "p": 7482, "t": 32400056093, "f": 123, "q": 100, "k": 1 }
    ]
  }
}
```

推奨同時選択ファイル:

- `main_Ticks.json`
- `main_Masters.json`
- `main_ItaRows.json`

読み込み仕様:

- `ticks` オブジェクトを銘柄コード別歩み値として読む。
- `map` がある場合、短縮キーを正式名に展開する。
- `main_Masters.json` 相当のマスタが同時選択されていれば銘柄名、取引日、終値、出来高などの情報に使う。
- `main_ItaRows.json` 相当の `ita` は現時点では受け流し。将来の実板スナップショット再生用。
- 複数銘柄は件数順に並び、ツールバーの銘柄セレクタで切り替える。
- `timestamp` が日中マイクロ秒/ミリ秒の相対値の場合は、取引日JSTの0:00を基準にUTC秒へ変換する。
- マスタの `timestamp` がepochマイクロ秒の場合はJST取引日をそこから推定する。
- マスタやファイル名から日付を取れない場合は、ファイルの更新日、最後に現在日付を使う。

実データ検証済み例:

```text
47 symbols / 9984 214201 ticks / 2026-06-01
```

## チャート仕様

時間足:

| キー | 秒数 | 表示 |
|---|---:|---|
| `2m` | 120 | 2分足 |
| `10m` | 600 | 10分足 |
| `15s` | 15 | 15秒足 |
| `1d` | 86400 | 日足 |

時刻仕様:

- 内部の `tick.time` は実UTC秒。
- Lightweight Charts表示用には `UTC + 9時間` の「表示用UTC」を渡す。
- 軸上の見え方をJSTに合わせるための処理。
- 日中足は09:00 JSTを基準にバケット化する。
- 日足はJST日付の0:00基準でバケット化する。

初期表示:

- リプレイ開始位置、つまりcursorが先頭の場合、日中足は09:00〜11:30を表示範囲にする。
- このため09:00時点では右側に前場終わりまでの空白が見える。
- 日足は全体表示。
- 2分足は直近約130本。
- 10分足は直近約90本。
- 15秒足は直近約280本。
- データ量が少ない場合は全体表示。

### 単一チャートモード

TradingViewに近い操作として、`Option + Enter` で単一チャートモードを切り替える。

仕様:

- 対象は最後にクリックしたチャート。
- 初期対象は15秒足。
- 単一チャートモードでは対象チャートだけをメイン領域全体に表示する。
- 他のチャート、日足、板、歩み値、リサイズハンドルは非表示にする。
- もう一度 `Option + Enter` を押すと通常レイアウトへ戻る。
- 切替後はチャートサイズを再計算する。

## ローソク足・出来高

通常のローソク足に加え、出来高を強調したCanvasレイヤーを持ちます。

出来高キャンドル仕様:

- `VolCandle` がONの場合、標準ローソク足を透明化し、Canvasで出来高強調足を描く。
- 表示範囲内の通常足から出来高上位15%を判定する。
- 出来高が多い足は太く、明るい緑/赤で表示する。
- 9:00と15:30は寄り/引け特別足として黄色で表示する。
- 特別足は通常出来高ランキングから除外する。

ツールチップ:

- 足種別
- 上昇/下降
- 始値終値差
- 出来高

## VWAP

- 日中足にVWAPを表示する。
- 日足にはVWAPを出さない。
- VWAPは09:00 JSTをセッション開始として日ごとにリセットする。
- `VWAP名` 設定でラインタイトル表示を切り替える。
- `vwapColor` で色を変更できる。

## 板・気配値パネル

現状の板は実板スナップショットではなく、読み込んだ歩み値全体から構成する「未来約定由来の検証板」です。

集計:

```text
total[price]     = その価格で当日約定する全数量
executed[price]  = cursor以前に約定済みの数量
remaining[price] = total - executed
```

分類:

- 現在値より上の価格: 売り気配側
- 現在値より下の価格: 買い気配側
- 現在値と同値: 現在値行

表示:

- 当日安値から高値までに上下30ティック分の余白を持たせる。
- 呼び値は価格差の最大公約数から自動推定する。
- 売買数量は横棒で相対表示する。
- 現在値行を中央に自動スクロールする。
- 水平線がある価格行は同色でハイライトする。
- 価格セルクリックで星マーカーを付ける。
- 十字カーソルの価格に近い行をアウトライン表示する。

下部統計:

- 現在値
- 現在時刻
- 売 `A / X`
- 買 `B / Y`

意味:

- `A`, `B`: 現在以降に残っている価格帯別数量
- `X`, `Y`: セッション全体でその側に存在した総数量
- 売買比率により `買い優勢`, `売り優勢`, `中立` を表示する。

## 歩み値パネル

- 現在カーソルまでの直近120件を表示する。
- 新しい約定が上に来る。
- 価格が前回より上なら `▲`、下なら `▼`、同値なら `─`。
- 表示列は時刻、価格、数量。
- ヘッダーの件数は現在カーソル位置までの件数。

## 描画ツール

### 水平線

- `水平線` ボタンまたは `H` キーでモード切替。
- チャートクリック位置の価格に水平線を追加。
- 線色と線種を指定できる。
- 追加した水平線は全チャートへ反映。
- 板価格行にも同色ハイライトが反映。

### レイライン

- `レイ` ボタンまたは `R` キーでモード切替。
- クリックした時間と価格から右方向へ線を伸ばす。
- 全チャートへ反映。

### 削除

- `削除` ボタンまたは `D` キー。
- RRがある場合はRRを削除。
- RRがない場合は直近の描画線を削除。

## RR計測

`RR計測` ボタンまたは `T` キーで開始します。

クリック順:

1. ENTRY
2. LC
3. TP

表示内容:

- LONG/SHORT判定
- ENTRY価格
- LC価格
- TP価格
- Risk
- Reward
- RR比
- ENTRY到達からTP到達までの時間
- ENTRY時刻
- TP時刻

仕様:

- ENTRY/LC/TPはレイとして描画される。
- 価格線付近をドラッグして価格を修正できる。
- ドラッグ時は価格だけでなく、そのX座標から開始時刻も更新する。
- TP到達時間は歩み値を先頭から探索して算出する。
- 設定でRR表示をツールバー内へ格納可能。

## 再生・ナビゲーション

内部状態:

- `ticks`: 現在銘柄の歩み値配列
- `cursor`: 現在の約定位置
- `autoPlay`: 再生タイマー

仕様:

- 再生間隔は50ms。
- 1回の進み幅は `Step`。
- `Step` は手入力できる。
- Step候補は1, 10, 50, 100, 500, 1000, 2000。
- `前場終` は現在カーソルの日付を基準に11:30:00以下の最後の約定へ移動する。
- `後場終` は現在カーソルの日付を基準に15:30:00以下の最後の約定へ移動する。
- `Z` / `X` の手動ステップ移動では、チャートの論理表示範囲を移動前後で保存・復元する。
- 復元時は `logicalRange` を優先し、取得できない場合のみ `timeRange` を使う。
- 手動ステップ描画中は一時的に右端追従を無効化し、描画直後と次フレームで表示範囲を戻す。
- そのため手動送りで右端のローソク足位置は動かない。
- 末尾で自動停止。
- CSV読み込み時、2日以内かつ2ファイル以内なら自動再生。
- 長いCSVは末尾表示。
- JSON銘柄読み込み時、2日以内なら自動再生。
- 銘柄セレクタで切り替えた場合は末尾表示。
